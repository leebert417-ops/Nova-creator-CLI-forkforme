# 人物卡生成指令（简要版）

## 生成要求

请根据以下简要模板生成人物卡，遵循以下规则：

1. 灵活选择：以下各项为可选内容，根据具体作品需求决定是否生成
2. 适度填写：不是所有字段都必须填写，根据角色重要性和故事需要选择性生成
3. 文件命名：输出文件保存到作品文件夹下，命名格式：`角色设定_角色名.xyaml`
4. 清洁输出：生成时不要包含注释内容
5. 额外要求：如用户有额外指示，优先遵循用户的具体要求

---

## 模板结构
```
<character_information character="名称">
基本资料：
  名称：[角色名称]
  性别：[性别]
  年龄：[年龄]
  身份：[职业、种族、社会身份等]
  背景：[简要的出身与关键经历]

外貌描述：
  [整体外貌特征的自然描述，包含体型、面部、发型、眼睛、肤色及显著特征等，形成完整的视觉印象]

服装描述：
  [角色的典型穿着风格描述，可包含日常装束、配饰及整体风格印象]

性格描述：
  [角色的核心性格特征、主要优缺点、典型行为习惯的综合描述]

关系：
  - 人物：{{user}}
    关系描述：[与{{user}}的关系]
  - 人物：[人物姓名]
    关系描述：[与该人物的关系]
</character_information>
```
