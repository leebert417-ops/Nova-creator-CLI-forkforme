# 作品润色指令

## 功能定义

对用户提供的文本进行润色，消除陈词滥调，提升描写的具体性和创意性，修正格式问题。

---

## 润色规则

### 1. 禁用词汇清单

以下词汇在润色时必须替换或改写：

#### 模糊量词
- 一丝、一抹、一丝丝、一般

#### 不确定性词汇
- 似乎、仿佛

#### 陈词滥调
- 野兽、小兽、幼兽
- 狡黠
- 不易察觉
- 甜腻
- 闪过、闪着、闪烁
- 低吼
- 该死的
- 发白、泛白
- 尖叫
- 灭顶

#### 老套表达
- 石子（常用于"如石子投入湖面"等意象）
- 涟漪（常用于心理描写）
- 四肢百骸

#### 特定俗套
- 他/她知道（用于心理描写时过于直白）
- 第三颗（纽扣，常见俗套桥段）
- 肉刃（色情描写中的陈词滥调）
- 尖锐（声音描写过于笼统）

### 2. 改写原则

#### A. 用具体替代模糊

**不佳示例**：
```
她眼中闪过一丝惊讶
他嘴角浮起一抹笑意
她的声音仿佛很轻
```

**优化示例**：
```
她的瞳孔微微收缩，眉梢扬起
他单侧嘴角上扬，露出半边牙齿
她的声音像羽毛拂过耳畔，轻得几乎听不见
```

#### B. 用动态替代静态

**不佳示例**：
```
他的指节泛白
她的眼中闪烁着光芒
```

**优化示例**：
```
他握紧拳头，指关节嵌入掌心，青筋暴起
她的眼睛亮了起来，瞳孔像被点燃的星辰
```

#### C. 用直接影响替代老套意象

**不佳示例**：
```
这句话如同石子投入湖面，在她心中荡起涟漪
```

**优化示例**：
```
这句话让她愣在原地，心跳骤然加速
这句话如同钝刀，在她胸口慢慢切开一道口子
```

#### D. 用创意表达替代俗套

**不佳示例**：
```
他低吼一声
她发出尖叫
该死的，他知道自己麻烦了
```

**优化示例**：
```
他喉咙深处挤出一声闷响
她的声音拔高了八度，在走廊里回荡
他咬紧后槽牙，心知这次栽了
```

### 3. 格式修正

#### A. 语言规范
- 统一使用中文简体
- 删除不适当的特殊符号
- 修正标点符号错误

#### B. 突兀内容
- 修正混入的繁体字
- 修正混入的外文（非专有名词）
- 修正网络黑话、拼音缩写（根据语境保留或改写）

## 润色示例

### 示例 1：心理描写

**原文**：
```
他的话让她心中泛起一丝涟漪，她似乎有些动摇了，眼中闪过一抹复杂的神色。她知道，自己不该再相信他。
```

**润色后**：
```
他的话让她停顿了一瞬，呼吸节奏微微错乱。她的眼神飘忽不定，在他脸上游移了几秒，最终落在地板上。她在心里反复告诫自己——不该再相信他。
```

### 示例 2：动作描写

**原文**：
```
他低吼一声，指节泛白地握紧拳头，眼中闪烁着愤怒的光芒。该死的，他知道自己被骗了。
```

**润色后**：
```
他喉咙里挤出一声闷响，拳头握得骨节嘎吱作响，青筋在手背上暴起。他咬紧后槽牙，胸腔里憋着一团火——这次被耍了。
```

### 示例 3：情绪描写

**原文**：
```
她仿佛被这句话击中了，眼眶泛红，发出一声尖叫，然后转身跑开了。
```

**润色后**：
```
这句话像一记耳光，她整个人僵在原地，眼眶瞬间充血。她的声音拔高到破音，随后转身冲出门外，脚步踉跄。
```

---

## 使用方式

1. 用户提供需要润色的文本
2. 按照上述规则进行改写
3. 保持原文的情节和人物行为不变
4. 仅优化表达方式和具体性
5. 输出润色后的完整文本

---

## 额外要求

- 如用户有特殊风格偏好（如保留某些表达、强调某种氛围），优先遵循用户要求
- 对于专有名词、角色台词等，不做修改
- 如需大段润色，可分段处理，确保风格统一
